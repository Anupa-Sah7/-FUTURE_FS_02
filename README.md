# 📊 Mini CRM - Lead Management System

> A simple CRM to manage client leads from website contact forms.

---

## 🎯 What is this?

This is a **Customer Relationship Management (CRM)** system that helps businesses:
- 📋 Track leads (potential customers)
- 🔄 Update lead status (New → Contacted → Converted)
- 📝 Add notes and schedule follow-ups
- 📊 View analytics and reports

---

## ✨ Features

- ✅ Add, edit, delete leads
- ✅ Change lead status (New / Contacted / Converted)
- ✅ Add notes and follow-up dates
- ✅ Search and filter leads
- ✅ Dashboard with stats and charts
- ✅ Dark mode toggle
- ✅ Export leads to CSV
- ✅ Secure admin login

## 🚀 How to Run

### Step 1: Install Required Software

1. **Node.js** → https://nodejs.org/
2. **XAMPP** → https://www.apachefriends.org/

### Step 2: Setup Database

1. Open XAMPP Control Panel
2. Click **"Start"** on MySQL
3. Click **"Admin"** to open phpMyAdmin
4. Create a new database named: `crm_db`

### Step 3: Start Backend

Open terminal/command prompt and run:

```bash

npm install
